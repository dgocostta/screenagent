# ScreenAgent AI

AI-Driven Multi-Tenant Sales & Logistics Engine for Retailers.

## Features
- **Multi-Tenant Admin Console:** Manage multiple businesses from one dashboard.
- **AI Sales Agent:** WhatsApp-style chat interface with decision-making logic.
- **Logistics Engine:** Real-time stock tracking across multiple warehouses.
- **Data Migration:** Automatic scraping and importing of products from eBay/Adverts.ie.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Firebase (Firestore & Storage)
- Lucide React (Icons)
- Framer Motion (Animations)

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
